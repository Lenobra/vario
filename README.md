# vario
STM32L476RG based vario for measuring vertical hight, speed with battery supply. 

By Leonard & Bastian

---------------------------------------------------
Wireing

A0 	- PA0 	- Battery
D15	- PB8	- I2C SCL
D14	- PB9	- I2C SDA
D6	- PB10	- Buzzer
